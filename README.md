# sheikh-chat

A real-time chat application with a React frontend and a Node.js backend.

## Tech Stack

- **Frontend:**
  - React
  - Vite
  - Ant Design
- **Backend:**
  - Node.js
  - Express
  - WebSockets (`ws`)

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm

### Installation and Running

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd sheikh-chat
   ```

2. **Backend Setup:**
   ```bash
   cd backend
   npm install
   npm start
   ```
   The backend server will start on port 3000.

3. **Frontend Setup:**
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```
   The frontend development server will start on a port specified by Vite (usually 5173).

4. **Access the application:**
   Open your browser and navigate to the address provided by the Vite development server.
